# ChemSolver BASIC-A

Built specifically for Chemical Engineering BASIC-A.

## Run locally

```bash
npm install
npm run dev
```

Then open:
http://localhost:5173/

## Included
- Unit converter
- SG to moles
- Composition conversion
- Tank balance
- Material balance
- Stoichiometry
- Ideal gas
- Raoult / humidity
- Antoine equation
- Energy balance
- Steam / enthalpy helper
- Dark mode
- Logo and validation rules
